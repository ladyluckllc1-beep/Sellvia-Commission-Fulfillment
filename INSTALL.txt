Process Sellvia orders using your pending commission balance instead of credit card or deposit.

== Description ==

Sellvia Commission Manager allows you to:

* View all your Sellvia orders in one dashboard
* Monitor your commission balance (pending, available, total)
* Process orders using commission balance with one click
* Automatic order status updates
* Secure API communication via Supabase

== Installation ==

AUTOMATIC INSTALLATION (Recommended):

1. Login to your WordPress admin: https://tophitsemporium.shop/wp-admin
2. Go to Plugins > Add New > Upload Plugin
3. Choose the wordpress-plugin.zip file
4. Click Install Now
5. Click Activate Plugin
6. Access via WordPress Admin > Sellvia Orders

MANUAL INSTALLATION:

1. Upload the 'sellvia-commission-manager' folder to '/wp-content/plugins/'
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click 'Sellvia Orders' in the admin menu

== Configuration ==

NO CONFIGURATION REQUIRED! The plugin is pre-configured with your React app URL.

If you need to change the React app URL:
1. Edit sellvia-commission-manager.php
2. Find line 56
3. Update the URL to your custom domain

== Usage ==

1. Login to WordPress Admin
2. Click "Sellvia Orders" in the left menu
3. View your commission balance
4. Click "Pay with Commission" next to any order
5. Order is automatically processed and commission deducted

== Frequently Asked Questions ==

Q: Do I need to configure API keys?
A: No! API keys are already configured securely in the backend.

Q: What if I don't have enough commission balance?
A: The system will show an error. You'll need to use credit card or wait for more commissions.

Q: Can I reverse a processed order?
A: No, once processed, orders cannot be reversed. Contact Sellvia support if needed.

Q: Is my data secure?
A: Yes! All API communication is encrypted and API keys are stored securely in Supabase.

This plugin integrates with:

The plugin is designed to be simple and maintenance-free. All business logic is handled in the React app and Supabase Edge Functions.
